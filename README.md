# dark-mode-extension
A simple Chrome extension that lets you toggle dark mode on any website.

 Simple Dark Mode is a lightweight Chrome extension that lets you turn any website into dark mode.

Features:
- Toggle dark mode on any page
- Works instantly
- Very small and fast
- No tracking or data collection

Installation:
1. Download the repository as a ZIP.
2. Extract the files.
3. Open Chrome and go to chrome://extensions.
4. Enable Developer Mode.
5. Click "Load unpacked" and select the extension folder.

Click the extension button and press "Toggle Dark Mode" to invert the page colors.
